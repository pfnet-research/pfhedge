PFHedge Documentation
=====================

PFHedge is a `PyTorch <https://pytorch.org>`_-based framework for `Deep Hedging <https://arxiv.org/abs/1802.03042>`_.

|

Install:

.. code-block:: none

    pip install pfhedge
